# Sistema de Reservas Hoteleras

## Estructura del Proyecto

- **/config** - Archivos de configuración
- **/assets** - CSS, JS, imágenes
- **/views** - Vistas HTML/PHP
- **/controllers** - Lógica de negocio
- **/models** - Modelos de datos
- **/admin** - Panel de administrador
- **/operador** - Panel de operador
- **/usuario** - Panel de usuario

## Instalación

1. Ejecutar crear_estructura.bat en htdocs
2. Importar base de datos SQL Server (ReservaHotel)
3. Configurar config/database.php con tu contraseña
4. Acceder a http://localhost
